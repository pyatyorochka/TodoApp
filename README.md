# Приложение, в котором можно создавать задачи.

## Функции
- Список задач
- Теги
- Форматирование текста

## Установка зависимостей
1. Python
3. Перейдите в каталог с проектом:
   ```
   cd путь
   ```
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Запустите:
   ```
   python main.py
   ```
